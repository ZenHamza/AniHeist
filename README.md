## рҹҡҖ Getting Started

### Prerequisites

- **Node.js** 20.x or later
- **npm** 10.x or later
- **Docker** (optional, for deployment)

### Local Development

```bash
# 1. Clone the repository
git clone https://github.com/ZenHamza/AniHeist.git
cd AniHeist

# 2. Install dependencies
npm install

# 3. Set up environment variables
cp .env.example .env.local
# Edit .env.local with your config

# 4. Start the development server
npm run dev

# 5. Open in browser
open http://localhost:3000
```

### Production Build

```bash
# Build for production
npm run build

# Start production server
npm start
```

## рҹҢҗ API Endpoints

### Cached Routes (Next.js API Routes)

| Endpoint | Description | Cache TTL |
|---|---|---|
| `GET /api/cached/search?q=` | Search anime by title | 30 days |
| `GET /api/cached/discover` | Advanced filtered discovery | 30 days |
| `GET /api/cached/anime/[id]` | Anime metadata | 30 days |
| `GET /api/cached/anime/[id]/episodes` | Episode list | 30 days |
| `GET /api/cached/trending` | Trending anime | 30 days |
| `GET /api/cached/popular` | Popular anime | 30 days |
| `GET /api/cached/newest` | Newest releases | 30 days |
| `GET /api/cached/top-rated` | Top rated anime | 30 days |
| `GET /api/cached/recent` | Recently updated | 30 days |
| `GET /api/cached/genre?genre=` | Genre search (AniList) | 30 days |

### Proxy Routes

| Endpoint | Description |
|---|---|
| `GET /api/proxy/[...path]` | Backend API proxy |
| `GET /api/proxy-hls` | HLS stream proxy |
| `GET /api/video` | Video proxy |
| `GET /api/video-proxy` | Video stream proxy |

### Discover Query Parameters

| Param | Type | Example |
|---|---|---|
| `sort` | `TRENDING_DESC`, `POPULARITY_DESC`, `SCORE_DESC` | `sort=SCORE_DESC` |
| `genres` | `string[]` (repeatable) | `genres=Action&genres=Comedy` |
| `season` | `WINTER`, `SPRING`, `SUMMER`, `FALL` | `season=SUMMER` |
| `season_year` | number | `season_year=2024` |
| `format` | `TV`, `MOVIE`, `ONA`, `OVA`, `SPECIAL` | `format=TV` |
| `status` | `RELEASING`, `FINISHED`, `NOT_YET_RELEASED` | `status=RELEASING` |
| `page` | number (default: 1) | `page=1` |
| `per_page` | number (max: 50) | `per_page=20` |

## рҹҗі Docker Deployment

### Build & Run

```bash
# Build the Docker image
docker build -t aniheist-frontend:latest -f Dockerfile .

# Run with docker-compose
docker compose -f docker-compose.aniheist.yml up -d --build frontend

# Or run standalone
docker run -d \
  --name aniheist-frontend \
  -p 3000:3000 \
  -e API_INTERNAL_URL=https://api.yourdomain.com \
  -e NEXTAUTH_SECRET=your-secret \
  -e NEXTAUTH_URL=https://yourdomain.com \
  aniheist-frontend:latest
```

### nginx Configuration

The included `nginx.aniheist.conf` handles:
- вң… Reverse proxy to Next.js
- вң… Static file serving
- вң… SSL termination (Let's Encrypt)
- вң… Security headers
- вң… Cache control
- вң… Gzip compression
